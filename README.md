# 🌱 Saarthi – Your Digital Companion

Saarthi is a wellness and productivity companion designed to help students and individuals stay focused, organized, and mentally balanced throughout the day.

Inspired by the role of **Lord Krishna as Saarthi (guide) to Arjuna**, this application aims to support users during challenging moments through productivity tools, mindfulness features, and an AI companion.

## 🚀 Live Demo

🔗 https://saarthicompanion.netlify.app/

---

## ✨ Features

### ✅ Task Management

* Create, edit, and delete tasks
* Mark tasks as completed
* Local Storage support for persistent data

### 📝 Daily Journal

* Write and save daily reflections
* Data persists using Local Storage

### ⏳ Pomodoro Timer

* Focus timer based on the Pomodoro technique
* Start, pause, and reset functionality

### 🧘 Meditation Session

* Multiple session durations
* Ambient sounds (Rain, Ocean, Forest)
* Guided voice instructions using Speech Synthesis
* Volume control support

### 💬 MindMate AI

* AI-powered emotional support companion
* Quick prompts for:

  * Feeling stressed
  * Feeling low
  * Need someone to talk to
  * Help with focus
  * Sleep-related concerns
* Built using the Gemini API

### 🌤 Weather Widget

* Real-time weather information
* Displays:

  * Temperature
  * City
  * Date
  * Live clock

### 💡 Motivation Quotes

* Inspirational quotes to encourage productivity and consistency

### 🎨 Modern UI

* Fully responsive design
* Mobile, Tablet and Desktop support
* Glassmorphism-inspired interface
* Custom splash screen experience

## 🎯 Project Motivation

Many students struggle with maintaining productivity while also managing stress and mental well-being.

Saarthi combines productivity tools and wellness features into a single platform, helping users stay organized, focused, and emotionally supported.
